# @Furqs7 Automated Content Engine

**AI-powered daily X/Twitter posts with backend integration. No CORS issues. Actually works.**

## 🚀 Quick Start

1. Deploy to Netlify (drag & drop or GitHub)
2. Add your Anthropic API key
3. Click "Generate Posts"
4. Copy and paste to X
5. Done!
